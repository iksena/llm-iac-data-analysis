- **AWS::ApiGateway::Deployment** - API deployment
- **AWS::ApiGateway::Stage** - API stage (e.g., 'prod')

The deployment follows this logical order:

1. **IAM Role** → Must exist before Lambda creation
2. **Lambda Function** → Depends on IAM Role
3. **RestApi** → Can be created independently
4. **Resource** → Depends on RestApi
5. **Method** → Depends on Resource
6. **Permission** → Depends on both Lambda and RestApi
7. **Deployment** → Depends on Method
8. **Stage** → Depends on Deployment

- **Logical ID**: HelloWorldFunction
