## 1. Key AWS Services Required

- **Subnets** (AWS::EC2::Subnet) - Public and private subnets (2 public, 2 private for high availability)
- **Route Tables** (AWS::EC2::RouteTable) - Public and private route tables
- **Routes** (AWS::EC2::Route) - Routes for internet traffic
- **Subnet Route Table Associations** (AWS::EC2::SubnetRouteTableAssociation) - Associates subnets with route tables
- **EC2 Instance** (AWS::EC2::Instance) - Web server instance
- **Security Group** (AWS::EC2::SecurityGroup) - Controls inbound/outbound traffic
- **IAM Role & Instance Profile** (AWS::IAM::Role, AWS::IAM::InstanceProfile) - For EC2 instance permissions
- **SSM Parameter Store** - For dynamic AMI retrieval

## 2. Properties and Dependencies

### Critical Dependencies:
1. VPC must be created before Internet Gateway
2. Internet Gateway must be attached to VPC before creating routes
3. Route tables must exist before associating subnets
4. Security Group must exist before creating EC2 instance
5. IAM Role must exist before creating Instance Profile

### Property Dependencies:
- Subnet CIDR blocks must be within VPC CIDR range
- Route destination must be valid (0.0.0.0/0 for internet)
- Security Group rules must specify valid protocols and ports
- EC2 instance must reference existing subnet, security group, and IAM profile

- **Private Subnet CIDRs**: `10.0.2.0/24`, `10.0.4.0/24`

### EC2 Configuration:
- **Instance Type**: `t3.micro` (free tier eligible, suitable for web server)
- **AMI**: Retrieved dynamically using SSM Parameter Store: `/aws/service/ami-amazon-linux-latest/amzn2-ami-hvm-x86_64-gp2`
- **KeyPair**: Parameter with default value 'default-key' (user can override)
- **Root Volume Size**: 8 GB (default for Amazon Linux 2)

### Security Group:
- **SSH Port**: 22 (for administration)
- **HTTP Port**: 80 (for web traffic)
- **HTTPS Port**: 443 (for secure web traffic)

### AMI Retrieval:


### Availability Zones:

```yaml
Fn::GetAZs: !Ref 'AWS::Region'
```

### KeyPair Parameter:

```yaml
Parameters:

    Type: String
    Default: default-key
    Description: Name of an existing EC2 KeyPair to enable SSH access
```
