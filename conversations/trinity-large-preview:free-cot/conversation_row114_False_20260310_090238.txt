# AWS B2B Data Interchange (B2BI) Pipeline Architecture Plan

1. **AWS B2B Data Interchange (B2B Integration)** - Core service for EDI processing
2. **Amazon S3** - Storage for input EDI files and output JSON data

4. **AWS IAM** - Service roles and permissions

6. **AWS CloudWatch** - Logging and monitoring

## Architecture Overview

The solution will create a complete B2B integration pipeline with:
- B2BI profile for partner configuration
- Transformer resource for EDI X12_214 to JSON mapping
- Input S3 bucket for receiving EDI files
- Output S3 bucket for storing transformed JSON
- IAM roles for service permissions
- Event-driven processing (S3 upload triggers transformation)

- **Resource Type**: `AWS::B2BIntegration::Profile`
- **Properties**:
  - Business contact details (name, email, phone)
  - Partner identification information
  - Communication protocols
  - Security settings

- **Resource Type**: `AWS::B2BIntegration::Transformer`
- **Properties**:
  - Mapping logic for X12_214 to JSON conversion
  - Schema definitions
  - Transformation rules
  - Error handling configurations

### 3. S3 Buckets
- **Input Bucket**: `AWS::S3::Bucket` for receiving EDI files
- **Output Bucket**: `AWS::S3::Bucket` for storing JSON data
- **Policies**: Bucket policies for access control
- **Events**: S3 event notifications for triggering processing

### 4. IAM Roles and Policies
- **Service Role**: For B2B Integration service access
- **Lambda Role** (if needed): For custom processing
- **S3 Access Policies**: For bucket permissions

### S3 Bucket Security
```yaml
Resources:
  InputBucket:
    Type: AWS::S3::Bucket
    Properties:
      BucketName: !Ref InputBucketName
      Versioning:
        Status: Enabled
      OwnershipControls:
        Rules:
          - ObjectOwnership: BucketOwnerEnforced
      BucketPolicy:
        Version: "2012-10-17"
        Statement:
          - Effect: Allow
            Principal:
              Service: b2bintegration.amazonaws.com
            Action: s3:PutObject
            Resource: !Sub "arn:aws:s3:::${InputBucketName}/*"
```
