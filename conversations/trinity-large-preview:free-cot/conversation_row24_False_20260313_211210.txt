- IAM role follows principle of least privilege

### Cost Optimization
- Default to t3.micro EC2 instances (free tier eligible)
- Default to cache.m3.medium ElastiCache instances (specified requirement)
- Implement proper cleanup in Lambda function
- Use appropriate instance counts and sizes based on requirements
