- **DelaySeconds** - Optional delay for message processing
- **MaximumMessageSize** - Optional size limit
- **MessageRetentionPeriod** - Optional retention duration
- **ReceiveMessageWaitTimeSeconds** - Optional long polling duration
- **VisibilityTimeout** - Optional message visibility timeout

- **MessageRetentionPeriod**: `345600` (4 days, AWS default)
